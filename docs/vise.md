# Vise

An essential feature of Vise is the ability to be used alongside other frameworks. In the following example we loaded Bootstrap and Vise to create two separate grid systems.

```html
<div class="container">
    <div class="row">
        <div></div>
        <div></div>
    </div>
</div>
<div class="vise">
    <div class="row">
        <div></div> 
        <div></div>
        <div ></div>      
    </div>
</div>
```
